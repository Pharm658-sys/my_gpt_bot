# Telegram GPT Bot

Простой Telegram-бот с интеграцией OpenAI GPT.

## Описание

- Команда /start  
- Кнопка "Задать вопрос GPT"  
- Пользователь вводит запрос  
- Бот отвечает с помощью GPT  

## Используемые технологии

- Python 3.10+  
- Aiogram 3  
- OpenAI API  
- Asyncio  

## Установка

1. Клонируйте репозиторий или сохраните скрипт.  
2. Установите зависимости: pip install aiogram openai python-dotenv
3. Создайте файл '.env' c переменными: (BOT_TOKEN) и (OPENAI_API_KEY)

## Использование

- Запустите бота в Telegram  
- Напишите /start  
- Нажмите кнопку "Задать вопрос GPT"  
- Введите свой вопрос  
- Получите ответ от GPT
